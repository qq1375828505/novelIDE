## 为什么选择网文写作IDE？

市面上的写作工具要么是纯文本编辑器，要么依赖云端服务。**网文写作IDE** 是第一个把 **IDE 工作流** + **AI 智能体** + **完全离线** 融为一体的网文创作工具。

| | 普通笔记App | 云端AI写作 | **网文写作IDE** |
|---|---|---|---|
| 离线写作 | ✅ | ❌ | ✅ |
| AI辅助 | ❌ | ✅ | ✅ |
| 资料管理 | ❌ | ❌ | ✅ 8种类型 |
| 人物关系图 | ❌ | ❌ | ✅ |
| 文章校对 | ❌ | 依赖网络 | ✅ 纯本地引擎 |
| EPUB导出 | ❌ | ❌ | ✅ |
| 隐私保护 | 看平台 | ❌ 数据上传 | ✅ 100%本地 |

## 全部功能

### 📝 写作 & 编辑

| 功能 | 说明 |
|------|------|
| **双编辑器** | 纯文本（Markdown）/ 富文本（加粗/标题/图片/链接）自由切换 |
| **全文搜索替换** | 跨章节全局搜索，黄色高亮，逐条/全部替换，自动更新字数 |
| **语音输入** | 中文语音转文字，支持标点识别 |
| **自动保存** | 1.5秒延迟防抖 + 退出强制保存，绝不丢稿 |
| **历史快照** | 每3分钟自动快照，保留最近20个版本 |
| **50步撤销** | 跨保存周期的操作历史栈 |
| **字数统计** | 实时字数 + 30天柱状图 + 连续打卡 + 每日目标进度条 |
| **章节数超限** | 超1万字黄色警告，超1万5千字建议拆章 |

### 🤖 AI 写作助手

#### AI 对话
- 多会话管理，历史记录保存到 SQLite
- 上下文压缩（超过40条自动压缩，避免 token 超限）
- **选择资料上下文** — 从资料库选择角色/设定/章节/伏笔，自动拼入对话
- **去AI味** — 一键将文本按 Humanizer 8大规则改写

#### 编辑器内 AI 操作
- 长按选中文字 → **润色 / 扩写 / 续写 / 联网查证**
- 联网搜索结果可一键保存到资料库

#### 精修与审查
- **一键精修** — 8个维度：语病检查 / 节奏分析 / 文风统一 / 冗余删除 / 对话优化 / 描写增强 / 钩子检测 / 战力一致性
- **全文审查** — 设定冲突 / 战力一致性 / 伏笔追踪 / 角色一致性
- **爽点密度报告** — 分析每章爽点分布
- **水文检测** — 检测凑字数、重复描写
- **爆款标题生成** — AI 生成多个候选标题

#### 自定义 Agent
- **Agent 市场** — 内置多种写作智能体
- **创建 Agent** — 自定义名称、图标、提示词、参数模板
- **去AI味 Agent** — 基于维基百科 AI 写作特征指南的 8 大规则
- **Agent 工具** — 35+ 工具执行器，覆盖读/写/编辑/删除/分析

#### 多模型 & 风格预设
- **兼容主流 AI** — 智谱GLM / DeepSeek / 通义千问 / 本地 Ollama 等
- **25个写作风格预设** — 都市/玄幻/穿越/悬疑/女频全覆盖
- **多模型切换** — 不同场景用不同模型

## 🏗️ 技术架构

| 层级 | Android (Flutter) | Windows (Electron) |
|------|-------------------|-------------------|
| **框架** | Flutter 3.32+ / Dart 3.8+ | Electron + TypeScript |
| **UI** | Material 3 + Riverpod | Tailwind CSS |
| **编辑器** | TextField + WebView（自研引擎） | Monaco Editor |
| **数据库** | SQLite (sqflite) | SQLite (better-sqlite3) |
| **配置** | Hive + flutter_secure_storage | electron-store |
| **网络** | Dio（兼容主流 AI 接口） | fetch/axios |
| **打包** | APK | NSIS / AppImage / DMG |

```
用户操作 → UI (Material 3 Widget)
    ↓
状态管理 (Riverpod Provider/StateNotifier)
    ↓
业务逻辑 (Service 层)
    ↓
数据层
├── SQLite (sqflite) — 作品/章节/卷/角色/统计
├── Hive — 主题/配置持久化
├── 文件系统 (path_provider) — 正文Markdown + 记忆文件
└── Secure Storage — API Key 加密
```

## 🛠️ 快速开始

### Android

```bash
git clone https://github.com/qq1375828505/novelIDE.git
cd novelIDE
flutter pub get
dart run build_runner build --delete-conflicting-outputs  # 首次必须
flutter run
```

### Windows

```bash
cd "win版/novel-ide-windows"
npm install
npm run dev
```

| 项目 | Android | Windows |
|------|---------|---------|
| SDK | Flutter >= 3.8.0 | Node.js >= 18 |
| 系统 | Android 6.0+ | Windows 10+ |
